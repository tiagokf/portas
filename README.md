# Scanner de Portas

Ferramenta online gratuita para verificar a disponibilidade de IPs e domínios. Teste a conectividade de servidores e descubra se estão acessíveis na internet.

## Funcionalidades

- Verificação de portas abertas em IPs ou domínios
- Interface moderna com design responsivo
- Resultados em tempo real com cores diferenciadas
- Validação de entrada e tratamento de erros
- SEO otimizado para indexação

## Tecnologias Utilizadas

- PHP
- Tailwind CSS
- HTML5
- JavaScript

## Instalação

1. Clone este repositório:
   ```bash
   git clone git@github.com:tiagokf/portas.git
   ```

2. Acesse o diretório do projeto:
   ```bash
   cd scanner-portas
   ```

3. Abra o arquivo `index.php` em seu servidor web (Apache/Nginx) com suporte a PHP.

## Uso

1. Acesse a página inicial no navegador.
2. Insira o IP ou domínio que deseja testar.
3. Informe as portas separadas por vírgula (ex: `22,80,443`).
4. Clique em "Verificar Portas" para obter os resultados.
